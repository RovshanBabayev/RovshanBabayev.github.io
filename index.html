Data cleansing

Nashville Housing Data

Data contains information of houses in Nashville city between years 2013 and 2019. The data includes Sale Dates, House sale prices, names of house owners, house characteristics and specific lands where the houses are located and other important insides.

The cleansing process has been done in MySql. Used cleansing operations on data : Modifying, updating, joinin tables, breaking out columns into seperate columns, changing column values with specific criterias, deleting dublicates.

Covid-19 Data

Data contains information on worldwide Covid-19 pandemic. The data is stored in daily format between 2020 January and 2022 October. Data contains Countries information,death,cases,vactinations and other important insides to analyze.

Analyzing processes have been done in MySql. Different methods have been used to get new tables with critical insides. Temperory tables has been created based on analysis for further data visualization usage.
